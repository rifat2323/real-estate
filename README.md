Make sure to configure your environment variables for the project to run smoothly. The required variables include:
- `MONGO_URI`
- `NEXT_PUBLIC_api_key`
- `NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME`
- `CLOUDINARY_API_SECRET`
- `NEXT_PUBLIC_URL`
- `ACCESS_KEY`
- `REFRESH_KEY`
